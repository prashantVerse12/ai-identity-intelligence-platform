A recruiter-grade AI Identity Intelligence Platform designed to analyze a user's professional digital footprint using a hybrid Explainable AI (XAI) and Generative AI architecture.

This platform combines:

* local machine learning inference
* NLP-based signal extraction
* explainable AI reasoning
* recruiter-focused career analytics
* generative strategic insights

to produce a unified professional identity analysis system.

The application consolidates:

* Resume/CV content
* LinkedIn summaries
* GitHub technical profiles
* Portfolio descriptions
* Skills and technical signals

into a single AI-driven intelligence engine.

# ⚡ Local Development Setup

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/ai-identity-intelligence-platform.git

cd ai-identity-intelligence-platform
```

---

## 2. Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### macOS/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Train ML Model

```bash
python train_model.py
```

This generates:

```text
models/career_model.joblib
```

---

## 5. Configure Environment Variables

Create:

```text
.env
```

Add:

```env
GEMINI_API_KEY=your_api_key_here
```

Get Gemini API Key:
https://aistudio.google.com/

---

## 6. Run Backend Server

```bash
python app.py
```

OR

```bash
uvicorn backend.main:app --host 0.0.0.0 --port 3000
```

Application:

```text
http://localhost:3000
```
